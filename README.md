
## Description

## Installation

```bash
$ pnpm install
```

## Running the app in Docker
To start all the services in development mode (detached mode):

```bash
# watch mode
$ pnpm run docker:up:dev
```

## Running the app 

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
